## 提高编码效率的小建议

- 好好配置一下ISE的editor，打开自动补全功能（虽然很鸡肋），能规避手抽带来的Bug
- 或者调用外部的编辑器，安装插件提高效率（具体可以搜索“ISE 第三方编辑器”）

## 魔改MARS

mars非常良心，解压以后可以看到所有源代码。

适当修改Mars，可以获得模拟器写入通用寄存器的信息。

与仿真输出做对比，有助于快速查找错误，精准定位。
